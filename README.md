# OOJ Lab Programs (Object Oriented Java)

This repository contains **8 Object Oriented Java (OOJ) lab programs**, implemented as part of the academic laboratory course.  
Each program demonstrates core **OOP concepts in Java** such as classes, objects, inheritance, polymorphism, exception handling, multithreading, packages, and interfaces.
